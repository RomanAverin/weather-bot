### .env variables 

| Variable | Description |
| ----------- | ----------- |
| *TELEGRAM_BOT_TOKEN* | Telegram bot token |
| *RUST_LOG* | Logging level (trace, debug, info, warning, error) |
| *GITHUB* | Link to the projects repository for the agent string |
